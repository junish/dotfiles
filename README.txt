INSTALL
    ドットファイルとbinをカレントディレクトリに反映させる
        # ./set-dotfiles.sh

    必要なパッケージをインストールする
        # set-yum.sh

    .vimを作成する
        # vim-update-bundles

    vimshellを使用しているため、vimprocをコンパイルする
        # pushd ~/.vim/bundle/vimproc/; make -f make_gcc.mak; popd
